mysql2sqlite by jjohn

Convert a MySQL schema generated by say MySQLWorkbench into a schema that can be used for sqlite

Several times when folks create proof of concept(POC) type applications, they use sqlite as it is very easy to use and small, unlike a MySQL installation. e.g. I may want to quickly crank out a POC on my laptop or netbook and in such situations, sqlite comes very handy. However even a POC needs a proper schema diagram to ensure that it functions properly. Schema diagrams force you to flush out details at design stage and it is a very useful and rewarding exercise to do even for a POC. Several tools are available to create schemas. One such tool is the MySQLWorkbench. However, the schema generated by the MySQLWorkbench works only on MySQL DB. It has several features which are absent in other DBs such as the InnoDB engine etc. This script will allow you to take a schema generated by MySQLWorkbench and 'prune' it, so that it can be used in sqlite.
